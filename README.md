# Проект для команды "ВКонтакте"  
**Разработчик: Горбунов Иван**  

## Описание проекта  
Приложение для управления данными с возможностью:  
- Просмотра данных в табличном формате  
- Добавления новых записей через форму  
- Бесконечной подгрузки данных при скролле  
- Валидации вводимых данных  

## Функциональность  
✅ Таблица с динамическими полями
✅ Infinite Loader для подгрузки данных  
✅ Форма создания записей с валидацией  
✅ Отправка данных через API  
✅ Интеграция с json-server  
✅ Обработка состояний загрузки и ошибок  

## Стек технологий  

### Основные технологии  
| Технология | Назначение |  
|------------|------------|  
| React 18 | Базовый фреймворк для UI |  
| TypeScript | Статическая типизация |  
| TanStack Query v5 | Управление состоянием асинхронных операций |  
| React Hook Form | Работа с формами и валидацией |  
| Axios | HTTP-клиент для работы с API |  
| Vite | Сборка проекта |  

### Почему выбран TanStack Query?  
1. **Для асинхронных операций**  
   - Оптимален для загрузки/отправки данных  
   - Встроенная обработка состояний (загрузка, ошибки, успех)  

2. **Встроенный кэш и инвалидация**  
   - Автоматическое кэширование запросов  
   - Инвалидация кэша при мутациях данных  
   - Фоновое обновление данных  

3. **Не нужен дополнительный стейт-менеджер**  
   - Локальный стейт управляется через useState  
   - Глобальное состояние приложения не требуется  
   - Упрощает архитектуру приложения  

### Почему выбран React Hook Form?  
1. **Оптимизированная работа с формами**  
   - Неконтролируемые компоненты  
   - Минимальные перерендеры  

2. **Валидация без перерендеров**  
   - Изоляция ошибок валидации  
   - Поддержка синхронной/асинхронной валидации  

3. **Простая интеграция с UI**  
   - Легкое подключение к любым компонентам  
   - Совместимость с UI-библиотеками  

## Запуск проекта  


### Установка  
1. Установка библиотеки:  
```bash  
npm install json-server
```  
2. Запустить JSON-server (база данных):  
```bash  
npx json-server db.json  
```  

3. Запустить приложение:  
```bash  
npm run dev  
```  

### Скрипты  
| Скрипт | Назначение |  
|--------|------------|  
| `npm run dev` | Запуск dev-сервера |  
| `npm run build` | Сборка production-версии |  
| `npx json-server db.json` | Запуск json-server |
